---
title: Glossar
description: Technische Begriffe im Mobilfunk
---

| Begriff | Erklärung |
| :--- | :--- |
| **Netzbetreiber** | Unternehmen mit eigener Funkinfrastruktur (Telekom, Vodafone, O2, 1&1) |
| **Provider** | Anbieter ohne eigenes Netz, nutzt Infrastruktur der Netzbetreiber |
| **5G** | Aktueller Mobilfunkstandard für hohe Datenraten und niedrige Latenz |
| **LTE / 4G** | Mobilfunkstandard der vierten Generation |
| **GSM / 2G** | Basis-Standard für Telefonie und SMS |
| **Prepaid** | Tarifmodell auf Guthabenbasis ohne feste Laufzeit |
| **Postpaid** | Tarifmodell mit monatlicher Abrechnung (Laufzeitvertrag) |
| **Roaming** | Nutzung des Mobilfunkanschlusses in fremden Netzen (z.B. Ausland) |
| **Drosselung** | Reduzierung der Datenübertragungsrate nach Verbrauch des Inklusivvolumens |
| **SIM-Karte** | Teilnehmer-Identitätsmodul zur Identifizierung im Mobilfunknetz |
| **eSIM** | Fest im Gerät verbauter, digital programmierbarer Chip statt physischer Karte |
| **Standalone (SA)** | 5G-Netz ohne Abhängigkeit von LTE-Infrastruktur |
| **National Roaming** | Nutzung eines inländischen Fremdnetzes durch Kooperation (z.B. 1&1 bei O2) |
| **MVNO** | Mobile Virtual Network Operator (Anbieter ohne eigenes Funknetz) |
| **MultiSIM** | Nutzung mehrerer SIM-Karten mit derselben Rufnummer/Vertrag |
